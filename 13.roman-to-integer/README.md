# 13. 罗马数字转整数 (Roman to Integer)

## 题目描述

罗马数字包含以下七种字符：

```text
I   表示 1
V   表示 5
X   表示 10
L   表示 50
C   表示 100
D   表示 500
M   表示 1000
```

例如：

- `2` 写作 `II`
- `12` 写作 `XII`
- `27` 写作 `XXVII`

通常情况下，罗马数字中小的数字在大的数字右边，表示相加。例如：

- `VI = 5 + 1 = 6`
- `XV = 10 + 5 = 15`

但也有特殊情况，表示减法：

- `IV = 5 - 1 = 4`
- `IX = 10 - 1 = 9`
- `XL = 50 - 10 = 40`
- `XC = 100 - 10 = 90`
- `CD = 500 - 100 = 400`
- `CM = 1000 - 100 = 900`

给定一个罗马数字字符串 `s`，将其转换成整数。

**示例 1：**

```text
输入：s = "III"
输出：3
```

**示例 2：**

```text
输入：s = "LVIII"
输出：58
解释：L = 50, V = 5, III = 3。
```

**示例 3：**

```text
输入：s = "MCMXCIV"
输出：1994
解释：M = 1000, CM = 900, XC = 90, IV = 4。
```

**提示：**

- `1 <= s.length <= 15`
- `s` 仅含字符 `('I', 'V', 'X', 'L', 'C', 'D', 'M')`
- 题目保证 `s` 是一个有效的罗马数字，且表示范围在 `[1, 3999]`

## 算法知识

### 哈希表 / 映射

本题首先需要建立罗马字符到整数的映射关系，例如：

- `I -> 1`
- `V -> 5`
- `X -> 10`
- `L -> 50`
- `C -> 100`
- `D -> 500`
- `M -> 1000`

有了这张映射表后，就可以把字符串中的每个字符快速转换为对应数值。

### 加法规则与减法规则

罗马数字的核心在于：

1. 如果当前字符对应的值大于等于后一个字符，通常做加法。
2. 如果当前字符对应的值小于后一个字符，说明出现了减法结构，此时应该做减法。

例如：

- `"VI"` 中 `V(5) > I(1)`，所以是 `5 + 1`
- `"IV"` 中 `I(1) < V(5)`，所以是 `5 - 1`

因此，这道题可以通过一次线性扫描完成。

## 解法：线性扫描（推荐）

### 思路

从左到右遍历字符串 `s`，对每个字符 `s[i]`：

1. 先取出它对应的数值 `value`
2. 如果它后面还有字符，并且 `value < 后一个字符的数值`
   - 说明当前字符属于减法结构
   - 应该把 `value` 从答案中减去
3. 否则
   - 把 `value` 加到答案中

这样遍历一遍后，就能得到最终答案。

### 为什么这样做是对的？

因为罗马数字里所有的特殊情况，本质上都是“小数在大数前面时表示减法”。

例如 `MCMXCIV`：

- `M = 1000`，后面是 `C(100)`，`1000 > 100`，所以加 `1000`
- `C = 100`，后面是 `M(1000)`，`100 < 1000`，所以减 `100`
- `M = 1000`，后面是 `X(10)`，所以加 `1000`
- `X = 10`，后面是 `C(100)`，所以减 `10`
- `C = 100`，后面是 `I(1)`，所以加 `100`
- `I = 1`，后面是 `V(5)`，所以减 `1`
- `V = 5`，最后一个字符，直接加 `5`

总和为：

```text
1000 - 100 + 1000 - 10 + 100 - 1 + 5 = 1994
```

### 复杂度分析

- **时间复杂度**：O(n)，其中 `n` 是字符串长度，只需遍历一次。
- **空间复杂度**：O(1)，映射表大小固定。

## 代码

### C++

```cpp
#include <unordered_map>
#include <string>
using namespace std;

class Solution {
public:
    int romanToInt(string s) {
        unordered_map<char, int> mp = {
            {'I', 1}, {'V', 5}, {'X', 10}, {'L', 50},
            {'C', 100}, {'D', 500}, {'M', 1000}
        };

        int ans = 0;
        for (int i = 0; i < s.size(); i++) {
            int value = mp[s[i]];
            if (i + 1 < s.size() && value < mp[s[i + 1]]) {
                ans -= value;
            } else {
                ans += value;
            }
        }

        return ans;
    }
};
```

### Python

```python
class Solution:
    def romanToInt(self, s: str) -> int:
        mp = {
            "I": 1, "V": 5, "X": 10, "L": 50,
            "C": 100, "D": 500, "M": 1000
        }

        ans = 0
        for i in range(len(s)):
            value = mp[s[i]]
            if i + 1 < len(s) and value < mp[s[i + 1]]:
                ans -= value
            else:
                ans += value

        return ans
```
