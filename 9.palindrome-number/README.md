# 9. 回文数 (Palindrome Number)

## 题目描述

给你一个整数 `x`，如果 `x` 是一个回文整数，返回 `true`；否则，返回 `false`。

**回文数** 是指正序（从左向右）和倒序（从右向左）读都一样的整数。

例如：

- `121` 是回文数
- `-121` 不是回文数，因为从右向左读是 `121-`
- `10` 不是回文数，因为倒过来读是 `01`

**示例 1：**

```text
输入：x = 121
输出：true
```

**示例 2：**

```text
输入：x = -121
输出：false
```

**示例 3：**

```text
输入：x = 10
输出：false
```

**提示：**

- `-2^31 <= x <= 2^31 - 1`

## 算法知识

### 回文的对称性

回文数的核心特征是左右对称。最直接的方法是把整数转成字符串，然后用双指针判断首尾字符是否相同。

不过这道题还可以只使用整数运算完成判断，不依赖额外的字符串空间，更符合算法题的训练目标。

### 数学反转数字

如果把整个整数完全反转，再和原数比较，也能判断是否回文。但这样有两个问题：

1. 可能发生整数溢出。
2. 没有必要把整个数都反转，只需要反转一半即可。

对于一个回文数：

- 左半部分应该等于右半部分反转后的结果。
- 如果数字个数是奇数，中间那一位不影响回文判断，可以直接去掉。

例如 `1221`：

- 原数不断去掉末位，最终左半部分会变成 `12`
- 反转后的右半部分会变成 `12`

例如 `12321`：

- 左半部分最后是 `12`
- 反转后的右半部分是 `123`
- 去掉中间位后，即 `123 / 10 = 12`，两边相等

## 解法：反转后一半数字（推荐）

### 思路

先处理几个可以直接判定的情况：

1. 如果 `x < 0`，一定不是回文数。
2. 如果 `x` 的个位是 `0`，但 `x` 不等于 `0`，也一定不是回文数，因为回文数不能以 `0` 开头。

然后使用数学方法反转数字的后一半：

1. 定义 `reversedHalf = 0`。
2. 每次取出 `x` 的最后一位，拼接到 `reversedHalf` 后面。
3. 同时把 `x` 去掉最后一位。
4. 当 `reversedHalf >= x` 时，说明已经处理完一半数字，可以停止。

最后分两种情况判断：

- 如果数字位数是偶数，应满足 `x == reversedHalf`
- 如果数字位数是奇数，应满足 `x == reversedHalf / 10`

因为奇数位的中间数字不影响回文性，所以需要把 `reversedHalf` 的最后一位去掉再比较。

### 为什么只反转一半？

这样做有两个好处：

1. 避免把整个整数都反转，逻辑更精炼。
2. 反转一半后的数值规模更小，不容易产生溢出风险。

### 复杂度分析

- **时间复杂度**：O(log10 n)，其中 `n` 是整数 `x` 的绝对值，循环次数等于数字位数的一半。
- **空间复杂度**：O(1)，只使用常数个变量。

## 代码

### C++

```cpp
class Solution {
public:
    bool isPalindrome(int x) {
        // 负数不是回文数；末位为 0 且本身不为 0 的数也不是回文数
        if (x < 0 || (x % 10 == 0 && x != 0)) {
            return false;
        }

        int reversedHalf = 0;
        while (x > reversedHalf) {
            reversedHalf = reversedHalf * 10 + x % 10;
            x /= 10;
        }

        // 偶数位：x == reversedHalf
        // 奇数位：x == reversedHalf / 10（去掉中间位）
        return x == reversedHalf || x == reversedHalf / 10;
    }
};
```

### Python

```python
class Solution:
    def isPalindrome(self, x: int) -> bool:
        # 负数不是回文数；末位为 0 且本身不为 0 的数也不是回文数
        if x < 0 or (x % 10 == 0 and x != 0):
            return False

        reversed_half = 0
        while x > reversed_half:
            reversed_half = reversed_half * 10 + x % 10
            x //= 10

        # 偶数位：x == reversed_half
        # 奇数位：x == reversed_half // 10（去掉中间位）
        return x == reversed_half or x == reversed_half // 10
```
