# 45. 跳跃游戏 II (Jump Game II)

## 题目描述

给定一个长度为 `n` 的整数数组 `nums`。初始位置在下标 `0`。

数组中的每个元素 `nums[i]` 表示你在该位置最多可以向前跳跃的长度。

你的目标是使用最少的跳跃次数到达数组的最后一个位置。

题目保证你总是可以到达最后一个位置。

**示例 1：**

```text
输入：nums = [2,3,1,1,4]
输出：2
解释：跳到最后一个位置的最小跳跃数是 2。
从下标 0 跳到下标 1，再跳 3 步到最后一个位置。
```

**示例 2：**

```text
输入：nums = [2,3,0,1,4]
输出：2
```

**提示：**

- `1 <= nums.length <= 10^4`
- `0 <= nums[i] <= 1000`

## 算法知识

### 贪心

这题要求“最少跳几次”，很多人一开始会想到动态规划。

但这题有一个更简洁也更高效的做法：贪心。

贪心的核心不是每次真的跳到某个具体位置，而是：

- 在当前这一步能够覆盖的范围内
- 统计下一步最远可以到哪里

当我们走完整个当前可覆盖区间时，就说明必须再跳一步了。

### 分层扫描的感觉

你可以把这题想成在按“层”往前推进：

1. 第 0 次跳跃能覆盖一段区间
2. 在这段区间里继续看，下一次跳跃最远能覆盖到哪里
3. 一旦扫完整个当前区间，就把跳跃次数加一

这种感觉有点像 BFS 的按层扩展，只不过这里我们不需要真的开队列。

## 解法：贪心维护当前边界和下一步最远位置（推荐）

### 思路

我们维护三个变量：

- `steps`：当前用了多少步
- `end`：当前这一步最远能覆盖到哪里
- `farthest`：在当前覆盖范围内，下一步最远能到哪里

然后从左到右遍历数组，但不需要遍历最后一个位置。

对于每个位置 `i`：

1. 计算从这里出发最远能跳到哪里：

```text
i + nums[i]
```

2. 用它更新：

```text
farthest = max(farthest, i + nums[i])
```

3. 如果当前已经走到了本层边界 `end`
   - 说明当前这一跳的覆盖范围已经扫描完了
   - 必须进入下一跳
   - 所以：
     - `steps++`
     - `end = farthest`

最后返回 `steps` 即可。

### 为什么走到 `end` 时才加一步

因为 `end` 表示：

- 当前用了 `steps` 步后，最多能覆盖到的位置

在到达 `end` 之前，我们仍然还在“当前这一步能到的范围里”活动。

只有当我们把这个范围都看完了，才知道：

- 下一步最远到底能扩到哪里

所以这时才真正需要：

```text
steps += 1
```

### 为什么不需要真的决定跳到哪个点

因为我们关心的是：

- 最少跳几次

并不是必须输出具体路径。

在当前区间里，到底落在哪个点不重要；
重要的是这个区间整体能帮我们把“下一层边界”推多远。

所以只维护：

- 当前边界
- 下一步最远边界

就够了。

### 以 `[2,3,1,1,4]` 为例

开始时：

- `steps = 0`
- `end = 0`
- `farthest = 0`

遍历：

1. `i = 0`
   - 最远能到 `0 + 2 = 2`
   - `farthest = 2`
   - 此时 `i == end`
   - 说明第一层扫描完了
   - `steps = 1`
   - `end = 2`

2. 继续扫描区间 `[1, 2]`
   - `i = 1` 时最远能到 `4`
   - `farthest = 4`

3. `i = 2`
   - 扫完当前层
   - `steps = 2`
   - `end = 4`

已经能覆盖最后一个位置，所以答案是 `2`。

### 复杂度分析

- **时间复杂度：** `O(n)`，只遍历数组一遍
- **空间复杂度：** `O(1)`，只使用常数额外变量

## 代码

### C++

```cpp
#include <vector>
using namespace std;

class Solution {
public:
    int jump(vector<int>& nums) {
        int steps = 0;
        int end = 0;
        int farthest = 0;

        for (int i = 0; i < nums.size() - 1; i++) {
            farthest = max(farthest, i + nums[i]);

            if (i == end) {
                steps++;
                end = farthest;
            }
        }

        return steps;
    }
};
```

### Python

```python
from typing import List


class Solution:
    def jump(self, nums: List[int]) -> int:
        steps = 0
        end = 0
        farthest = 0

        for i in range(len(nums) - 1):
            farthest = max(farthest, i + nums[i])

            if i == end:
                steps += 1
                end = farthest

        return steps
```
