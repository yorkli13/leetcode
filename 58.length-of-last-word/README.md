# 58. 最后一个单词的长度 (Length of Last Word)

## 题目描述

给你一个字符串 `s`，字符串由若干单词和空格组成。
请返回字符串中最后一个单词的长度。

这里的单词是指仅由字母组成、不包含空格的最大子字符串。

**示例 1：**

```text
输入：s = "Hello World"
输出：5
解释：最后一个单词是 "World"，长度为 5
```

**示例 2：**

```text
输入：s = "   fly me   to   the moon  "
输出：4
解释：最后一个单词是 "moon"，长度为 4
```

**示例 3：**

```text
输入：s = "luffy is still joyboy"
输出：6
解释：最后一个单词是 "joyboy"，长度为 6
```

**提示：**

- `1 <= s.length <= 10^4`
- `s` 仅由英文字母和空格 `' '` 组成
- `s` 中至少存在一个单词

## 算法知识

### 字符串逆向扫描

这类题最自然的思路之一，是从字符串末尾往前扫描。

原因很直接：

- 我们要找的是“最后一个单词”
- 字符串尾部可能有很多空格
- 先从后往前跳过空格，再继续统计字母个数，就正好能拿到答案

这种方法的好处是：

1. 不需要额外拆分字符串
2. 不需要额外数组存单词
3. 逻辑很贴合题目本身

## 解法：从后往前跳过空格再计数（推荐）

### 思路

我们用一个指针 `i` 从字符串最后一个位置开始往前走。

整个过程分两步：

1. 先跳过末尾所有空格
2. 再继续往前数最后一个单词有多少个字符

### 第一步：跳过尾部空格

如果字符串最后有空格，比如：

```text
"Hello World   "
```

那这些空格都不属于最后一个单词，所以要先跳过去：

```text
while i >= 0 and s[i] == ' '
```

直到找到最后一个非空格字符。

### 第二步：统计最后一个单词长度

当 `i` 停在最后一个单词的末尾后，就继续向前走，直到遇到空格或到达字符串开头：

```text
while i >= 0 and s[i] != ' '
```

每走一步，计数器加一。

最后这个计数器就是答案。

### 为什么这样做是对的

因为最后一个单词一定满足：

- 它右边要么是字符串结尾
- 要么右边全是空格

所以先删掉尾部空格后，最右边那一段连续字母就是最后一个单词。

而我们只统计这一段的长度，就刚好得到答案。

### 以示例 2 为例

```text
s = "   fly me   to   the moon  "
```

从后往前看：

1. 先跳过最后两个空格
2. 指针停在 `n` 上
3. 继续往前数：`n -> o -> o -> m`
4. 遇到前面的空格停止

一共数了 4 个字符，所以答案是：

```text
4
```

### 复杂度分析

- **时间复杂度：** `O(n)`，最坏情况下需要扫描整个字符串
- **空间复杂度：** `O(1)`，只使用常数个额外变量

## 代码

### C++

```cpp
#include <string>
using namespace std;

class Solution {
public:
    int lengthOfLastWord(string s) {
        int i = s.size() - 1;
        int len = 0;

        while (i >= 0 && s[i] == ' ') {
            i--;
        }

        while (i >= 0 && s[i] != ' ') {
            len++;
            i--;
        }

        return len;
    }
};
```

### Python

```python
class Solution:
    def lengthOfLastWord(self, s: str) -> int:
        i = len(s) - 1
        length = 0

        while i >= 0 and s[i] == ' ':
            i -= 1

        while i >= 0 and s[i] != ' ':
            length += 1
            i -= 1

        return length
```
