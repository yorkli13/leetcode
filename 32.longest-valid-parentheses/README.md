# 32. 最长有效括号 (Longest Valid Parentheses)

## 题目描述

给你一个只包含 `'('` 和 `')'` 的字符串，找出最长有效（格式正确且连续）括号子串的长度。

**示例 1：**

```text
输入：s = "(()"
输出：2
解释：最长有效括号子串是 "()"
```

**示例 2：**

```text
输入：s = ")()())"
输出：4
解释：最长有效括号子串是 "()()"
```

**示例 3：**

```text
输入：s = ""
输出：0
```

**提示：**

- `0 <= s.length <= 3 * 10^4`
- `s[i]` 为 `'('` 或 `')'`

## 算法知识

### 栈

本题虽然也是括号问题，但和第 20 题“有效的括号”不同：

- 第 20 题只需要判断是否合法
- 本题要找的是“最长连续合法子串的长度”

因此，我们不仅要知道括号是否匹配，还要知道匹配后这段合法区间的边界。

栈非常适合保存括号的**下标**，因为：

- 遇到 `'('` 时，可以把它的位置压栈
- 遇到 `')'` 时，可以尝试与最近的 `'('` 匹配
- 一旦匹配成功，就能根据当前下标和“上一个未匹配位置”计算合法区间长度

### 边界下标

本题中一个关键技巧是：

```text
栈里先放入 -1
```

这个 `-1` 不是合法字符位置，而是一个“边界哨兵”。

它的作用是：

1. 方便计算从开头开始的有效括号长度
2. 当出现无法匹配的 `')'` 时，可以把它当作新的边界位置

## 解法：栈保存下标（推荐）

### 思路

维护一个栈，栈中存的是下标，不是字符。

初始化时先压入：

```cpp
-1
```

然后遍历字符串：

1. 如果当前字符是 `'('`
   - 直接把当前下标压栈
2. 如果当前字符是 `')'`
   - 先弹出栈顶，表示尝试匹配一个 `'('`
   - 如果弹出后栈为空
     - 说明当前 `')'` 没有可匹配的左括号
     - 把当前下标压入栈，作为新的边界
   - 如果弹出后栈不为空
     - 说明形成了一个合法区间
     - 当前合法长度就是：

```text
当前下标 i - 栈顶下标
```

然后更新答案最大值。

### 为什么 `i - 栈顶` 就是当前长度？

因为栈顶保存的是“当前这段合法括号之前，最近一个没有被匹配的位置”。

所以：

- 从 `栈顶 + 1`
- 到当前下标 `i`

这一整段就是一个连续有效括号区间。

长度自然就是：

```text
i - 栈顶
```

### 为什么这样做可行？

每次遇到 `')'` 时：

- 如果能匹配成功，栈里剩下的栈顶就是这一段合法区间左边最近的边界
- 如果匹配失败，就把当前下标作为新的边界

因此在扫描过程中，我们始终能正确知道“当前合法区间从哪里开始”，并实时更新最长长度。

### 复杂度分析

- **时间复杂度**：O(n)，其中 `n` 是字符串长度，每个下标最多进栈出栈一次。
- **空间复杂度**：O(n)，最坏情况下栈中可能存放所有下标。

## 代码

### C++

```cpp
#include <stack>
#include <string>
#include <algorithm>
using namespace std;

class Solution {
public:
    int longestValidParentheses(string s) {
        stack<int> st;
        st.push(-1);  // 哨兵，表示合法区间左边界之前的位置
        int ans = 0;

        for (int i = 0; i < s.size(); i++) {
            if (s[i] == '(') {
                st.push(i);
            } else {
                st.pop();
                if (st.empty()) {
                    st.push(i);  // 当前右括号无法匹配，作为新的边界
                } else {
                    ans = max(ans, i - st.top());
                }
            }
        }

        return ans;
    }
};
```

### Python

```python
class Solution:
    def longestValidParentheses(self, s: str) -> int:
        stack = [-1]  # 哨兵，表示合法区间左边界之前的位置
        ans = 0

        for i, ch in enumerate(s):
            if ch == "(":
                stack.append(i)
            else:
                stack.pop()
                if not stack:
                    stack.append(i)  # 当前右括号无法匹配，作为新的边界
                else:
                    ans = max(ans, i - stack[-1])

        return ans
```
