# 44. 通配符匹配 (Wildcard Matching)

## 题目描述

给你一个输入字符串 `s` 和一个模式字符串 `p`，请你实现支持 `'?'` 和 `'*'` 的通配符匹配。

匹配规则如下：

- `'?'` 可以匹配任何单个字符
- `'*'` 可以匹配任意长度的字符串，包括空字符串

要求匹配必须覆盖整个字符串，而不是部分匹配。

**示例 1：**

```text
输入：s = "aa", p = "a"
输出：false
解释："a" 无法匹配整个 "aa"
```

**示例 2：**

```text
输入：s = "aa", p = "*"
输出：true
解释："*" 可以匹配任意字符串
```

**示例 3：**

```text
输入：s = "cb", p = "?a"
输出：false
解释："?" 可以匹配 "c"，但后面的 "a" 无法匹配 "b"
```

**提示：**

- `0 <= s.length, p.length <= 2000`
- `s` 只包含小写英文字母
- `p` 只包含小写英文字母、`'?'` 和 `'*'`

## 算法知识

### 动态规划

这题要判断：

- `s` 的前一部分
- 和 `p` 的前一部分

是否能匹配。

这类“前缀和前缀之间的匹配关系”问题，通常很适合动态规划。

我们可以定义：

```text
dp[i][j] = s 的前 i 个字符 和 p 的前 j 个字符 是否匹配
```

这样就能把大问题拆成更小的子问题。

### `?` 和 `*` 的区别

这题的关键是弄清楚两个特殊字符的含义：

1. `?`
   - 只能匹配 1 个字符
2. `*`
   - 可以匹配 0 个字符
   - 也可以匹配 1 个字符
   - 还可以匹配多个字符

也就是说：

- `?` 是“单字符通配”
- `*` 是“任意长度通配”

## 解法：二维动态规划（推荐）

### 思路

定义：

```text
dp[i][j]
```

表示：

- `s` 的前 `i` 个字符
- 是否能和 `p` 的前 `j` 个字符

完全匹配。

### 初始状态

1. `dp[0][0] = true`
   - 空字符串和空模式可以匹配

2. 当 `s` 是空串时
   - 只有当前面的模式全是 `*`，才可能匹配空串

所以：

```text
如果 p[j - 1] == '*'
则 dp[0][j] = dp[0][j - 1]
```

### 转移规则

#### 情况 1：当前模式字符是普通字符或 `?`

如果：

- `p[j - 1] == s[i - 1]`
- 或者 `p[j - 1] == '?'`

那么当前这一位可以匹配，此时只要看前面的部分是否匹配：

```text
dp[i][j] = dp[i - 1][j - 1]
```

#### 情况 2：当前模式字符是 `*`

`*` 有两种理解方式：

1. 把 `*` 当成空串
   - 那就相当于忽略这个 `*`
   - 所以看：

```text
dp[i][j - 1]
```

2. 把 `*` 当成匹配了一个或多个字符
   - 那么 `s[i - 1]` 这个字符就交给 `*` 来吞掉
   - 模式 `p` 还停在当前 `*`
   - 所以看：

```text
dp[i - 1][j]
```

综合起来：

```text
dp[i][j] = dp[i][j - 1] || dp[i - 1][j]
```

### 为什么这个 DP 是对的

因为每一步都只关心：

- 当前最后一个字符怎么处理

如果最后一个模式字符是普通字符或 `?`，问题就缩成前一个前缀。

如果最后一个模式字符是 `*`，那它不是匹配空串，就是匹配掉当前字符串最后一个字符。

这两种情况已经完整覆盖了 `*` 的所有可能。

### 复杂度分析

- **时间复杂度：** `O(m * n)`，其中 `m = s.length()`，`n = p.length()`
- **空间复杂度：** `O(m * n)`，用于 DP 表

## 代码

### C++

```cpp
#include <vector>
#include <string>
using namespace std;

class Solution {
public:
    bool isMatch(string s, string p) {
        int m = s.size(), n = p.size();
        vector<vector<bool>> dp(m + 1, vector<bool>(n + 1, false));

        dp[0][0] = true;

        for (int j = 1; j <= n; j++) {
            if (p[j - 1] == '*') {
                dp[0][j] = dp[0][j - 1];
            }
        }

        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                if (p[j - 1] == s[i - 1] || p[j - 1] == '?') {
                    dp[i][j] = dp[i - 1][j - 1];
                } else if (p[j - 1] == '*') {
                    dp[i][j] = dp[i][j - 1] || dp[i - 1][j];
                }
            }
        }

        return dp[m][n];
    }
};
```

### Python

```python
class Solution:
    def isMatch(self, s: str, p: str) -> bool:
        m, n = len(s), len(p)
        dp = [[False] * (n + 1) for _ in range(m + 1)]

        dp[0][0] = True

        for j in range(1, n + 1):
            if p[j - 1] == "*":
                dp[0][j] = dp[0][j - 1]

        for i in range(1, m + 1):
            for j in range(1, n + 1):
                if p[j - 1] == s[i - 1] or p[j - 1] == "?":
                    dp[i][j] = dp[i - 1][j - 1]
                elif p[j - 1] == "*":
                    dp[i][j] = dp[i][j - 1] or dp[i - 1][j]

        return dp[m][n]
```
