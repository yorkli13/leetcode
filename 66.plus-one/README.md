# 66. 加一 (Plus One)

## 题目描述

给定一个由整数数组 `digits` 表示的大整数，其中每个 `digits[i]` 是这个整数的第 `i` 位数字。

这些数字按从高位到低位排列，数组中不含前导零。

请在这个整数的基础上加 1，并返回结果数组。

**示例 1：**

```text
输入：digits = [1,2,3]
输出：[1,2,4]
解释：数组表示整数 123，加 1 后得到 124
```

**示例 2：**

```text
输入：digits = [4,3,2,1]
输出：[4,3,2,2]
解释：数组表示整数 4321，加 1 后得到 4322
```

**示例 3：**

```text
输入：digits = [9]
输出：[1,0]
```

**提示：**

- `1 <= digits.length <= 100`
- `0 <= digits[i] <= 9`
- `digits` 不包含任何前导零

## 算法知识

### 数组模拟进位

这题本质上就是模拟我们手算十进制加法时的“进位”过程。

比如：

```text
129 + 1 = 130
```

我们是从最低位开始加：

1. 先看最后一位
2. 如果没有进位，直接结束
3. 如果产生进位，就继续往前传

数组正好可以用来模拟这个过程。

## 解法：从低位向高位模拟进位（推荐）

### 思路

因为加的是 `1`，所以真正受影响的一定是最低位，也就是数组最后一个元素。

我们从后往前扫描 `digits`。

### 情况 1：当前位不是 9

如果当前位小于 9，比如：

```text
... 3
```

那么加 1 以后只会变成：

```text
... 4
```

不会产生进位。

这时直接：

```text
digits[i]++
```

然后整个问题就结束了，可以立刻返回答案。

### 情况 2：当前位是 9

如果当前位是 9，那么加 1 后就会变成 10。

在十进制里，这意味着：

- 当前位变成 0
- 向前一位继续进 1

所以代码里会写成：

```text
digits[i] = 0
```

然后继续往前处理。

### 为什么如果所有位都是 9，要在最前面补一个 1

例如：

```text
999 + 1 = 1000
```

如果所有位置都是 9，那么我们从后往前处理之后会得到：

```text
[0,0,0]
```

但这还少了最前面的那个 `1`。

所以最后如果整轮扫描都没有提前返回，说明整个数组原来全是 9。

这时就新建一个结果：

```text
[1,0,0,0]
```

也就是：

1. 开头补一个 1
2. 后面跟原长度个 0

### 以 `[1,2,9]` 为例

从后往前看：

1. 最后一位是 `9`
   - 变成 `0`
   - 向前进位
2. 前一位是 `2`
   - 加 1 变成 `3`
   - 不再进位，直接结束

结果变成：

```text
[1,3,0]
```

### 以 `[9,9,9]` 为例

从后往前看：

1. 第 3 位 `9 -> 0`
2. 第 2 位 `9 -> 0`
3. 第 1 位 `9 -> 0`

全部处理完后还是有进位没放下，所以结果应该是：

```text
[1,0,0,0]
```

### 复杂度分析

- **时间复杂度：** `O(n)`，最坏情况下需要从后往前扫完整个数组
- **空间复杂度：** `O(1)`，如果不算返回结果的额外空间；只有全 9 时会构造一个新数组

## 代码

### C++

```cpp
#include <vector>
using namespace std;

class Solution {
public:
    vector<int> plusOne(vector<int>& digits) {
        for (int i = digits.size() - 1; i >= 0; i--) {
            if (digits[i] < 9) {
                digits[i]++;
                return digits;
            }
            digits[i] = 0;
        }

        vector<int> ans(digits.size() + 1, 0);
        ans[0] = 1;
        return ans;
    }
};
```

### Python

```python
from typing import List


class Solution:
    def plusOne(self, digits: List[int]) -> List[int]:
        for i in range(len(digits) - 1, -1, -1):
            if digits[i] < 9:
                digits[i] += 1
                return digits
            digits[i] = 0

        return [1] + [0] * len(digits)
```
