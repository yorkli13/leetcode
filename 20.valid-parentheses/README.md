# 20. 有效的括号 (Valid Parentheses)

## 题目描述

给定一个只包括 `'('`、`')'`、`'{'`、`'}'`、`'['`、`']'` 的字符串 `s`，判断字符串是否有效。

有效字符串需要满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。
3. 每个右括号都必须有一个对应的同类型左括号。

**示例 1：**

```text
输入：s = "()"
输出：true
```

**示例 2：**

```text
输入：s = "()[]{}"
输出：true
```

**示例 3：**

```text
输入：s = "(]"
输出：false
```

**示例 4：**

```text
输入：s = "([])"
输出：true
```

**提示：**

- `1 <= s.length <= 10^4`
- `s` 仅由括号 `'()[]{}'` 组成

## 算法知识

### 栈

栈（Stack）是一种“后进先出”的数据结构，也叫 LIFO（Last In, First Out）。

常见操作有：

- `push`：把元素压入栈顶
- `pop`：弹出栈顶元素
- `top` / `peek`：查看栈顶元素

本题非常适合用栈，因为括号匹配天然符合“最近打开的括号，必须最先关闭”的规律。

例如字符串：

```text
([{}])
```

匹配顺序一定是：

- 先匹配 `{}` 
- 再匹配 `[]`
- 最后匹配 `()`

也就是说，最后出现的左括号要最先被匹配掉，这正是栈的特点。

### 括号匹配

遍历字符串时：

- 遇到左括号，就先记下来，等待将来匹配
- 遇到右括号，就必须和当前“最近的那个未匹配左括号”配对

如果：

- 栈为空，却遇到右括号
- 栈顶左括号和当前右括号类型不匹配

那么字符串一定无效。

## 解法：栈模拟匹配（推荐）

### 思路

从左到右遍历字符串中的每个字符：

1. 如果是左括号 `(`、`[`、`{`
   - 直接压入栈中
2. 如果是右括号 `)`、`]`、`}`
   - 先检查栈是否为空
   - 如果为空，说明没有可匹配的左括号，返回 `false`
   - 否则取出栈顶左括号，判断类型是否匹配
   - 如果不匹配，返回 `false`

遍历结束后：

- 如果栈为空，说明所有括号都成功匹配，返回 `true`
- 如果栈不为空，说明还有未闭合的左括号，返回 `false`

### 为什么这样做可行？

因为括号必须按正确顺序闭合，所以每次遇到右括号时，它只能和“最近那个还没被匹配的左括号”配对。

这个“最近未匹配”的要求正好对应栈顶元素，因此用栈可以准确模拟整个匹配过程。

### 复杂度分析

- **时间复杂度**：O(n)，其中 `n` 是字符串长度，每个字符最多进栈和出栈一次。
- **空间复杂度**：O(n)，最坏情况下所有字符都是左括号，需要全部入栈。

## 代码

### C++

```cpp
#include <stack>
#include <string>
using namespace std;

class Solution {
public:
    bool isValid(string s) {
        stack<char> st;

        for (char ch : s) {
            if (ch == '(' || ch == '[' || ch == '{') {
                st.push(ch);
            } else {
                if (st.empty()) {
                    return false;
                }

                char top = st.top();
                st.pop();

                if ((ch == ')' && top != '(') ||
                    (ch == ']' && top != '[') ||
                    (ch == '}' && top != '{')) {
                    return false;
                }
            }
        }

        return st.empty();
    }
};
```

### Python

```python
class Solution:
    def isValid(self, s: str) -> bool:
        stack = []

        for ch in s:
            if ch in "([{":
                stack.append(ch)
            else:
                if not stack:
                    return False

                top = stack.pop()
                if (ch == ")" and top != "(") or \
                   (ch == "]" and top != "[") or \
                   (ch == "}" and top != "{"):
                    return False

        return not stack
```
