# 67. 二进制求和 (Add Binary)

## 题目描述

给你两个二进制字符串 `a` 和 `b`，请你返回它们的和，并用二进制字符串表示。

**示例 1：**

```text
输入：a = "11", b = "1"
输出："100"
```

**示例 2：**

```text
输入：a = "1010", b = "1011"
输出："10101"
```

**提示：**

- `1 <= a.length, b.length <= 10^4`
- `a` 和 `b` 仅由字符 `'0'` 或 `'1'` 组成
- 字符串如果不是 `"0"`，就不含前导零

## 算法知识

### 字符串模拟加法

这题和第 66 题“加一”本质是同一路思路，只不过：

- 第 66 题是在十进制数组上加 `1`
- 这一题是在二进制字符串上做两个数相加

核心仍然是：

1. 从最低位开始算
2. 维护一个进位 `carry`
3. 每一位算出当前结果
4. 再继续处理更高位

## 解法：从后往前模拟二进制加法（推荐）

### 思路

我们用两个指针：

- `i` 指向字符串 `a` 的最后一位
- `j` 指向字符串 `b` 的最后一位

再用一个变量：

- `carry` 表示当前进位

然后从后往前不断计算。

### 每一轮怎么算

在某一轮里，我们先把当前能取到的值都加起来：

```text
sum = carry
```

如果 `a[i]` 存在，就把它对应的二进制位加进去。

如果 `b[j]` 存在，也把它加进去。

所以当前总和可能是：

- `0`
- `1`
- `2`
- `3`

### 当前位和进位怎么拆

二进制加法和十进制加法一样，都是“当前位 + 进位”。

我们可以这样处理：

```text
当前结果位 = sum % 2
新的进位 = sum / 2
```

因为二进制里：

- `0 / 2` -> 当前位 `0`，进位 `0`
- `1 / 2` -> 当前位 `1`，进位 `0`
- `2 / 2` -> 当前位 `0`，进位 `1`
- `3 / 2` -> 当前位 `1`，进位 `1`

这正好符合二进制加法规则。

### 为什么最后要反转字符串

因为我们是从低位往高位算的。

也就是说，得到结果的顺序其实是：

- 先得到最低位
- 再得到更高位

但正常数字书写是高位在前，所以我们通常先把每一位追加到答案末尾，最后再整体反转一次。

### 以示例 2 为例

```text
a = "1010"
b = "1011"
```

从后往前计算：

1. `0 + 1 + carry 0 = 1`
   - 当前位 `1`
   - 进位 `0`
2. `1 + 1 + carry 0 = 2`
   - 当前位 `0`
   - 进位 `1`
3. `0 + 0 + carry 1 = 1`
   - 当前位 `1`
   - 进位 `0`
4. `1 + 1 + carry 0 = 2`
   - 当前位 `0`
   - 进位 `1`
5. 两个字符串都结束了，但还有进位 `1`
   - 再补一个 `1`

倒着得到的是：

```text
"10101"
```

反转后还是：

```text
"10101"
```

### 复杂度分析

- **时间复杂度：** `O(max(m, n))`，其中 `m` 和 `n` 是两个字符串长度
- **空间复杂度：** `O(max(m, n))`，用于保存结果字符串

## 代码

### C++

```cpp
#include <string>
#include <algorithm>
using namespace std;

class Solution {
public:
    string addBinary(string a, string b) {
        int i = a.size() - 1;
        int j = b.size() - 1;
        int carry = 0;
        string ans;

        while (i >= 0 || j >= 0 || carry) {
            int sum = carry;

            if (i >= 0) {
                sum += a[i] - '0';
                i--;
            }
            if (j >= 0) {
                sum += b[j] - '0';
                j--;
            }

            ans.push_back(sum % 2 + '0');
            carry = sum / 2;
        }

        reverse(ans.begin(), ans.end());
        return ans;
    }
};
```

### Python

```python
class Solution:
    def addBinary(self, a: str, b: str) -> str:
        i = len(a) - 1
        j = len(b) - 1
        carry = 0
        ans = []

        while i >= 0 or j >= 0 or carry:
            total = carry

            if i >= 0:
                total += int(a[i])
                i -= 1
            if j >= 0:
                total += int(b[j])
                j -= 1

            ans.append(str(total % 2))
            carry = total // 2

        return "".join(reversed(ans))
```
