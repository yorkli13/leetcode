# 8. 字符串转换整数 (atoi)

## 题目描述

请你实现一个 `myAtoi(string s)` 函数，使其能将字符串转换成一个 32 位有符号整数。

函数需要按照如下规则读取字符串：

1. 丢弃字符串开头的空格。
2. 检查下一个字符是否为正号 `'+'` 或负号 `'-'`，如果存在则记录符号。
3. 继续读取后续连续的数字字符，直到遇到非数字字符或字符串结束。
4. 将读取到的数字转换为整数，并根据符号得到最终结果。
5. 如果结果超过 32 位有符号整数范围 `[-2^31, 2^31 - 1]`，则截断到边界值。

如果没有读取到任何数字，则返回 `0`。

**示例 1：**

```text
输入：s = "42"
输出：42
解释：读取连续数字 "42"，转换为整数 42。
```

**示例 2：**

```text
输入：s = "   -42"
输出：-42
解释：先跳过前导空格，再读取负号和数字。
```

**示例 3：**

```text
输入：s = "4193 with words"
输出：4193
解释：读取到空格时停止，后面的字符被忽略。
```

**示例 4：**

```text
输入：s = "words and 987"
输出：0
解释：第一个非空字符不是数字或符号，无法进行有效转换。
```

**示例 5：**

```text
输入：s = "-91283472332"
输出：-2147483648
解释：结果小于 32 位有符号整数下界，截断为 -2^31。
```

**提示：**

- `0 <= s.length <= 200`
- `s` 由英文字母、数字、空格、`'+'`、`'-'` 和 `'.'` 组成
- 结果需要限制在 32 位有符号整数范围内：`[-2^31, 2^31 - 1]`

## 算法知识

### 字符串模拟

字符串模拟题通常要求按照题目给定的规则逐字符处理。关键不是数据结构复杂，而是**状态和边界**是否处理完整。

本题可以把转换过程拆成几个阶段：

1. 跳过前导空格。
2. 读取可选符号。
3. 连续读取数字。
4. 遇到第一个不合法字符时停止。
5. 根据 32 位整数范围进行截断。

这类题要特别注意：符号只能出现在数字之前，数字开始后遇到任何非数字字符都要停止，不能继续向后寻找新的数字。

### 溢出判断

32 位有符号整数范围为：

- `INT_MIN = -2147483648`
- `INT_MAX = 2147483647`

为了避免整数溢出，可以在每次拼接新数字之前判断：

- 若 `ans > INT_MAX / 10`，则再乘 10 一定溢出。
- 若 `ans == INT_MAX / 10` 且下一位数字超过边界末位，也会溢出。

因为负数下界的绝对值比正数上界大 1，所以判断时要区分正负边界。实现上可以像 C++ 代码一样按带符号结果累积，也可以像 Python 代码一样先按绝对值累积，再根据符号选择允许的上界。

## 解法：模拟解析字符串（推荐）

### 思路

使用下标 `i` 从左到右扫描字符串：

1. 跳过所有前导空格。
2. 如果当前字符是 `'+'` 或 `'-'`，记录符号 `sign`，然后后移一位。
3. 初始化结果 `ans = 0`。
4. 当当前字符是数字时，将其转换成数字 `digit`，尝试拼接到结果中。
5. 在拼接前判断是否会超过当前符号允许的边界：
   - 正数边界为 `2147483647`
   - 负数边界按绝对值计算为 `2147483648`
6. 如果超过边界，直接返回对应的 `INT_MAX` 或 `INT_MIN`。
7. 扫描结束后返回最终结果；如果按绝对值累积，则返回 `sign * ans`，如果按带符号结果累积，则直接返回 `ans`。

### 边界处理

- 空字符串或全是空格：返回 `0`。
- 第一个非空字符不是符号或数字：返回 `0`。
- 只有符号没有数字：返回 `0`。
- 数字后出现字母、空格、小数点等非数字字符：立即停止。
- 超过 32 位范围：按题意截断。

### 复杂度分析

- **时间复杂度**：O(n)，其中 n 是字符串长度，每个字符最多访问一次。
- **空间复杂度**：O(1)，只使用常数个变量。

## 代码

### C++

```cpp
#include <string>
#include <climits>
using namespace std;

class Solution {
public:
    int myAtoi(string s) {
        int n = s.size();
        int i = 0;

        // 1. 跳过前导空格
        while (i < n && s[i] == ' ') {
            i++;
        }

        // 2. 读取符号
        int sign = 1;
        if (i < n && (s[i] == '+' || s[i] == '-')) {
            sign = (s[i] == '-') ? -1 : 1;
            i++;
        }

        int ans = 0;

        // 3. 连续读取数字
        while (i < n && s[i] >= '0' && s[i] <= '9') {
            int digit = s[i] - '0';

            // 4. 拼接前判断是否会超过边界
            if (sign == 1 &&
                (ans > INT_MAX / 10 || (ans == INT_MAX / 10 && digit > 7))) {
                return INT_MAX;
            }
            if (sign == -1 &&
                (ans < INT_MIN / 10 || (ans == INT_MIN / 10 && digit > 8))) {
                return INT_MIN;
            }

            ans = ans * 10 + sign * digit;
            i++;
        }

        return ans;
    }
};
```

### Python

```python
class Solution:
    def myAtoi(self, s: str) -> int:
        INT_MIN, INT_MAX = -2**31, 2**31 - 1
        n = len(s)
        i = 0

        # 1. 跳过前导空格
        while i < n and s[i] == " ":
            i += 1

        # 2. 读取符号
        sign = 1
        if i < n and s[i] in "+-":
            sign = -1 if s[i] == "-" else 1
            i += 1

        ans = 0
        limit = INT_MAX if sign == 1 else -INT_MIN

        # 3. 连续读取数字
        while i < n and s[i].isdigit():
            digit = ord(s[i]) - ord("0")

            # 4. 拼接前判断是否会超过边界
            if ans > limit // 10 or (ans == limit // 10 and digit > limit % 10):
                return INT_MAX if sign == 1 else INT_MIN

            ans = ans * 10 + digit
            i += 1

        return sign * ans
```
