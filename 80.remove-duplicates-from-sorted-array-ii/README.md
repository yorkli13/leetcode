# 80. 删除有序数组中的重复项 II (Remove Duplicates from Sorted Array II)

## 题目描述

给你一个按非递减顺序排列的整数数组 `nums`，请你原地删除部分重复项，使得每个元素最多出现两次。

元素的相对顺序应该保持一致。

返回删除后数组的新长度 `k`。

你需要保证 `nums` 的前 `k` 个元素是最终保留下来的结果，数组后面的元素不重要。

**示例 1：**

```text
输入：nums = [1,1,1,2,2,3]
输出：5, nums = [1,1,2,2,3,_]
解释：函数应返回新长度 5，前五个元素为 1, 1, 2, 2, 3
```

**示例 2：**

```text
输入：nums = [0,0,1,1,1,1,2,3,3]
输出：7, nums = [0,0,1,1,2,3,3,_,_]
解释：函数应返回新长度 7，前七个元素为 0, 0, 1, 1, 2, 3, 3
```

**提示：**

- `1 <= nums.length <= 3 * 10^4`
- `-10^4 <= nums[i] <= 10^4`
- `nums` 已按非递减顺序排列

## 算法知识

### 双指针

这题和第 26 题很像，都是有序数组上的原地去重。

区别在于：

- 第 26 题：每个数字最多保留 1 次
- 第 80 题：每个数字最多保留 2 次

数组已经有序，所以相同的数字一定连续出现。

我们可以用一个慢指针 `slow` 表示当前结果区的长度，再用快指针 `fast` 扫描原数组。

每看到一个新数字，就判断它能不能写入结果区。

## 解法：保留最多两个重复元素（推荐）

### 思路

因为每个数字最多允许保留两次，所以前两个元素一定可以保留。

从第三个元素开始，判断当前元素 `nums[fast]` 是否可以放入结果区。

关键判断是：

```text
nums[fast] != nums[slow - 2]
```

其中 `slow` 表示当前已经保留了多少个元素。

如果当前数字和结果区倒数第二个数字不同，说明当前数字最多只出现了一次，可以继续保留。

如果当前数字和结果区倒数第二个数字相同，说明结果区里已经有两个相同数字了，当前这个就不能再保留。

### 为什么看 `slow - 2`

因为最多允许两个相同数字。

假设结果区最后两个位置已经是：

```text
1, 1
```

现在又看到一个 `1`，如果继续放进去，就会变成：

```text
1, 1, 1
```

超过两次。

而判断：

```text
nums[fast] == nums[slow - 2]
```

正好说明当前数字和倒数第二个保留数字相同。

由于数组有序，这就意味着当前数字如果加入，会成为第三个相同数字，所以要跳过。

### 具体流程

用 `slow` 表示下一个可以写入的位置。

初始化：

```text
slow = 0
```

然后从左到右遍历 `nums`。

对于当前数字 `num`：

- 如果 `slow < 2`
  - 说明结果区还不到两个元素，直接保留
- 否则，如果 `num != nums[slow - 2]`
  - 说明保留它不会导致某个数字出现超过两次
  - 可以写入 `nums[slow]`
- 否则跳过

每保留一个数字，就执行：

```text
nums[slow] = num
slow++
```

最后 `slow` 就是新长度。

### 以示例 1 为例

```text
nums = [1,1,1,2,2,3]
```

扫描过程：

1. 第一个 `1`，结果区不足两个元素，保留：`[1]`
2. 第二个 `1`，结果区不足两个元素，保留：`[1,1]`
3. 第三个 `1`，和 `nums[slow - 2]` 相同，跳过
4. 第一个 `2`，和结果区倒数第二个 `1` 不同，保留：`[1,1,2]`
5. 第二个 `2`，和结果区倒数第二个 `1` 不同，保留：`[1,1,2,2]`
6. `3`，和结果区倒数第二个 `2` 不同，保留：`[1,1,2,2,3]`

最终长度是：

```text
5
```

### 这套写法怎么推广

如果题目变成每个数字最多保留 `k` 次，那么判断可以改成：

```text
slow < k || nums[fast] != nums[slow - k]
```

这题里 `k = 2`，所以就是：

```text
slow < 2 || nums[fast] != nums[slow - 2]
```

### 复杂度分析

- **时间复杂度：** `O(n)`，只遍历一次数组
- **空间复杂度：** `O(1)`，原地修改，只使用常数额外变量

## 代码

### C++

```cpp
#include <vector>
using namespace std;

class Solution {
public:
    int removeDuplicates(vector<int>& nums) {
        int slow = 0;

        for (int fast = 0; fast < nums.size(); fast++) {
            if (slow < 2 || nums[fast] != nums[slow - 2]) {
                nums[slow] = nums[fast];
                slow++;
            }
        }

        return slow;
    }
};
```

### Python

```python
from typing import List


class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        slow = 0

        for fast in range(len(nums)):
            if slow < 2 or nums[fast] != nums[slow - 2]:
                nums[slow] = nums[fast]
                slow += 1

        return slow
```
