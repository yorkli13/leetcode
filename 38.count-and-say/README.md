# 38. 外观数列 (Count and Say)

## 题目描述

给定一个正整数 `n`，输出外观数列的第 `n` 项。

外观数列从字符串 `"1"` 开始，每一项都是对前一项的“读法”。

读法规则是：

- 连续出现了几个相同字符
- 然后这个字符本身是什么

例如：

- `"1"` 读作 “一个 1”，所以得到 `"11"`
- `"11"` 读作 “两个 1”，所以得到 `"21"`
- `"21"` 读作 “一个 2，一个 1”，所以得到 `"1211"`

**示例 1：**

```text
输入：n = 1
输出："1"
```

**示例 2：**

```text
输入：n = 4
输出："1211"
解释：
第 1 项是 "1"
第 2 项是 "11"
第 3 项是 "21"
第 4 项是 "1211"
```

**提示：**

- `1 <= n <= 30`

## 算法知识

### 模拟

这题没有复杂的数据结构，本质上是按题目规则一项一项生成。

所谓模拟，就是：

1. 完全按照题目描述的过程去做
2. 不去强行套别的高级算法
3. 把“规则”翻译成代码中的步骤

这题的规则非常明确：

- 从左到右扫描一个字符串
- 把连续相同字符分成一组
- 对每一组输出“个数 + 字符”

所以最自然的方法就是模拟。

### 双指针分组统计

在扫描字符串时，我们经常会遇到“连续一段相同字符”的情况。

这时可以用两个位置来理解：

- 一个位置表示这一组从哪里开始
- 另一个位置不断向后移动，直到找到不同字符

这样就能知道这一组：

- 是什么字符
- 一共有几个

## 解法：逐项生成 + 分组统计（推荐）

### 思路

我们从第 1 项 `"1"` 开始。

然后重复做 `n - 1` 次：

1. 读取当前字符串 `current`
2. 从左到右扫描它
3. 把连续相同字符统计成一组
4. 把这一组转换成：
   - 出现次数
   - 字符本身
5. 拼接成下一项字符串 `next`
6. 更新 `current = next`

最后返回 `current` 即可。

### 以 `"1211"` 为例

扫描 `"1211"`：

1. 第 1 组是 `"1"`
   - 1 个 1
   - 记成 `"11"`
2. 第 2 组是 `"2"`
   - 1 个 2
   - 记成 `"12"`
3. 第 3 组是 `"11"`
   - 2 个 1
   - 记成 `"21"`

拼起来就是：

```text
"11" + "12" + "21" = "111221"
```

所以 `"1211"` 的下一项是 `"111221"`。

### 为什么这样做是对的

因为题目本身要求的就是：

> 每一项由前一项“读出来”

而“读出来”的过程，本质就是把前一项按连续字符分组，然后把每组写成“数量 + 字符”。

所以我们只要老老实实模拟这个过程，就能得到正确答案。

### 复杂度分析

设第 `n` 项字符串长度为 `m`。

- **时间复杂度：** `O(m)` 到 `O(总生成长度)`，因为每一项都会被线性扫描
- **空间复杂度：** `O(m)`，用于构造下一项字符串

在 LeetCode 这题的范围 `n <= 30` 下，这个方法完全足够。

## 代码

### C++

```cpp
#include <string>
using namespace std;

class Solution {
public:
    string countAndSay(int n) {
        string current = "1";

        for (int step = 2; step <= n; step++) {
            string next;

            for (int i = 0; i < current.size(); ) {
                int j = i;
                while (j < current.size() && current[j] == current[i]) {
                    j++;
                }

                int count = j - i;
                next += to_string(count);
                next += current[i];
                i = j;
            }

            current = next;
        }

        return current;
    }
};
```

### Python

```python
class Solution:
    def countAndSay(self, n: int) -> str:
        current = "1"

        for _ in range(2, n + 1):
            next_str = []
            i = 0

            while i < len(current):
                j = i
                while j < len(current) and current[j] == current[i]:
                    j += 1

                count = j - i
                next_str.append(str(count))
                next_str.append(current[i])
                i = j

            current = "".join(next_str)

        return current
```
