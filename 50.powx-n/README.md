# 50. Pow(x, n)

## 题目描述

实现 `pow(x, n)`，即计算 `x` 的整数 `n` 次幂。

**示例 1：**

```text
输入：x = 2.00000, n = 10
输出：1024.00000
```

**示例 2：**

```text
输入：x = 2.10000, n = 3
输出：9.26100
```

**示例 3：**

```text
输入：x = 2.00000, n = -2
输出：0.25000
解释：2^-2 = 1 / 2^2 = 1 / 4 = 0.25
```

**提示：**

- `-100.0 < x < 100.0`
- `-2^31 <= n <= 2^31 - 1`
- `n` 是一个整数
- `x != 0` 或 `n > 0`

## 算法知识

### 快速幂

如果直接把 `x` 连乘 `n` 次，时间复杂度是 `O(n)`。

但这题的 `n` 可能非常大，所以需要更快的方法。

快速幂的核心思想是：

```text
x^8 = (x^4)^2
x^4 = (x^2)^2
x^2 = (x^1)^2
```

也就是说：

- 每次把底数平方
- 每次把指数减半

这样指数会很快缩小到 0，复杂度就能降到 `O(log n)`。

### 二进制拆分幂次

任意整数都可以写成二进制形式。

例如：

```text
13 = 1101(二进制) = 8 + 4 + 1
```

所以：

```text
x^13 = x^(8+4+1) = x^8 * x^4 * x^1
```

快速幂其实就是一边平方生成：

- `x`
- `x^2`
- `x^4`
- `x^8`

一边根据指数的二进制位，决定要不要把当前结果乘进去。

## 解法：迭代快速幂（推荐）

### 思路

先处理负指数：

- 如果 `n < 0`
- 那么：

```text
x^n = 1 / x^(-n)
```

所以可以先把 `n` 转成正数来算。

然后用快速幂：

1. 维护答案 `ans = 1`
2. 维护当前底数 `x`
3. 当指数 `n` 还大于 0 时：
   - 如果当前 `n` 是奇数，就把当前底数乘到答案里
   - 然后把底数平方
   - 再把指数右移一位，相当于除以 2

直到指数变成 0，答案就算完了。

### 为什么奇数时要乘到答案里

因为当指数的二进制最低位是 `1` 时，说明当前这一份幂次需要被选中。

例如：

```text
13 = 1101
```

最低位是 `1`，说明需要乘上当前这份底数。

之后：

- 底数平方
- 指数右移

继续处理下一位。

### 为什么要把 n 转成 long long

因为：

```text
n = -2^31
```

时，直接取反会溢出。

例如 32 位整数中：

```text
-2147483648
```

没有对应的正数 `2147483648` 可以放进 `int`。

所以要先转成更大范围的 `long long` 再处理。

### 复杂度分析

- **时间复杂度：** `O(log n)`，每次都把指数减半
- **空间复杂度：** `O(1)`，只用了常数额外变量

## 代码

### C++

```cpp
class Solution {
public:
    double myPow(double x, int n) {
        long long N = n;
        if (N < 0) {
            x = 1 / x;
            N = -N;
        }

        double ans = 1.0;

        while (N > 0) {
            if (N % 2 == 1) {
                ans *= x;
            }
            x *= x;
            N /= 2;
        }

        return ans;
    }
};
```
