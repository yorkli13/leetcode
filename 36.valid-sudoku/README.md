# 36. 有效的数独 (Valid Sudoku)

## 题目描述

给你一个 `9 x 9` 的数独板 `board`，判断这个数独当前状态是否有效。

只需要判断已经填入的数字是否满足数独规则，不需要判断这个数独是否一定有解。

数独规则如下：

1. 每一行中，数字 `1-9` 不能重复
2. 每一列中，数字 `1-9` 不能重复
3. 每一个 `3 x 3` 的小宫格中，数字 `1-9` 不能重复

注意：

- 数独板中的 `'.'` 表示空格
- 只有已经填入的数字需要检查

**示例 1：**

```text
输入：
board =
[["5","3",".",".","7",".",".",".","."]
,["6",".",".","1","9","5",".",".","."]
,[".","9","8",".",".",".",".","6","."]
,["8",".",".",".","6",".",".",".","3"]
,["4",".",".","8",".","3",".",".","1"]
,["7",".",".",".","2",".",".",".","6"]
,[".","6",".",".",".",".","2","8","."]
,[".",".",".","4","1","9",".",".","5"]
,[".",".",".",".","8",".",".","7","9"]]
输出：true
```

**示例 2：**

```text
输入：
board =
[["8","3",".",".","7",".",".",".","."]
,["6",".",".","1","9","5",".",".","."]
,[".","9","8",".",".",".",".","6","."]
,["8",".",".",".","6",".",".",".","3"]
,["4",".",".","8",".","3",".",".","1"]
,["7",".",".",".","2",".",".",".","6"]
,[".","6",".",".",".",".","2","8","."]
,[".",".",".","4","1","9",".",".","5"]
,[".",".",".",".","8",".",".","7","9"]]
输出：false
解释：左上角的 3x3 宫格里出现了两个 8。
```

**提示：**

- `board.length == 9`
- `board[i].length == 9`
- `board[i][j]` 是数字 `1-9` 或 `'.'`

## 算法知识

### 哈希集合 / 状态记录

这题的本质不是“解数独”，而是“检查重复”。

只要我们在遍历棋盘时，能够记录：

1. 某一行里哪些数字已经出现过
2. 某一列里哪些数字已经出现过
3. 某一个 `3 x 3` 宫格里哪些数字已经出现过

那么当我们再次遇到同一个数字时，就能立刻知道当前数独无效。

这种“边遍历边记录是否出现过”的做法，常用的数据结构就是哈希集合，特点是：

- 插入快
- 查找快
- 很适合做“去重判断”

## 解法：一次遍历 + 三类状态记录（推荐）

### 思路

我们遍历整个 `9 x 9` 棋盘。

对于每一个位置 `(i, j)`：

1. 如果当前是 `'.'`
   - 说明这个位置还没填数字，直接跳过
2. 如果当前是数字字符，比如 `'5'`
   - 检查它是否已经在第 `i` 行出现过
   - 检查它是否已经在第 `j` 列出现过
   - 检查它是否已经在对应的 `3 x 3` 宫格中出现过
3. 只要有一个地方重复
   - 立即返回 `false`
4. 如果都没重复
   - 就把这个数字记入对应的行、列、宫格

当整个棋盘都检查完还没有冲突时，返回 `true`。

### 3 x 3 宫格编号怎么计算

一个位置 `(i, j)` 属于哪个 `3 x 3` 宫格，可以这样算：

```text
box = (i / 3) * 3 + (j / 3)
```

例如：

- `(0, 0)` 在第 0 个宫格
- `(1, 5)` 在第 1 个宫格
- `(7, 8)` 在第 8 个宫格

因为总共刚好有 9 个宫格，所以我们可以像处理行、列一样处理它们。

### 为什么这个方法合适

因为题目只要求判断“当前是否有效”，并不要求把空格补完。

所以我们不需要回溯，也不需要搜索解，只需要把已经出现的数字检查一遍即可。

### 复杂度分析

- **时间复杂度：** `O(81)`，也就是 `O(1)`，因为棋盘大小固定为 `9 x 9`
- **空间复杂度：** `O(81)`，也就是 `O(1)`，因为最多记录固定数量的状态

## 代码

### C++

```cpp
#include <vector>
#include <unordered_set>
using namespace std;

class Solution {
public:
    bool isValidSudoku(vector<vector<char>>& board) {
        vector<unordered_set<char>> rows(9), cols(9), boxes(9);

        for (int i = 0; i < 9; i++) {
            for (int j = 0; j < 9; j++) {
                char ch = board[i][j];
                if (ch == '.') {
                    continue;
                }

                int box = (i / 3) * 3 + (j / 3);

                if (rows[i].count(ch) || cols[j].count(ch) || boxes[box].count(ch)) {
                    return false;
                }

                rows[i].insert(ch);
                cols[j].insert(ch);
                boxes[box].insert(ch);
            }
        }

        return true;
    }
};
```

### Python

```python
from typing import List


class Solution:
    def isValidSudoku(self, board: List[List[str]]) -> bool:
        rows = [set() for _ in range(9)]
        cols = [set() for _ in range(9)]
        boxes = [set() for _ in range(9)]

        for i in range(9):
            for j in range(9):
                ch = board[i][j]
                if ch == ".":
                    continue

                box = (i // 3) * 3 + (j // 3)

                if ch in rows[i] or ch in cols[j] or ch in boxes[box]:
                    return False

                rows[i].add(ch)
                cols[j].add(ch)
                boxes[box].add(ch)

        return True
```
